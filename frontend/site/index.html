Manage technical debt with machine learning

Heupr is an open source service that offers a variety of features including bug assignment, task labeling, and issue duplicate identification to help manage your GitHub projects.

Join the mailing list here to stay up-to-date on product features and releases or contribute to the codebase on GitHub!

Features

Assignment
Too many issues piling up? Heupr automatically assigns contributors to review new issues and stay ahead of backlog.

Labeling
Can't figure out what's what? Heupr parses and categorizes new issues by placing labels that you specify on them.

Duplicates
Feel like you're seeing double? Heupr identifies when an issue is raised that duplicates one previously opened. COMING SOON!
