Features

Assignment
Too many issues piling up? Heupr automatically assigns contributors to review new issues and stay ahead of backlog.

Labeling
Can't figure out what's what? Heupr parses and categorizes new issues by placing labels that you specify on them.

Duplicates
Feel like you're seeing double? Heupr identifies when an issue is raised that duplicates one previously opened.

Setup

Install the app on your GitHub repositories.

Add a .heupr.toml file to each repository.

Sit back and let Heupr do all the work.
